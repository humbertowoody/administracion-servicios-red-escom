Práctica 5 - Monitoreo de una Interfaz de Red

En esta práctica se realizaron una serie de rutas que permiten obtener información sobre la interfaz de red predeterminada.

  • /conteo: Muestra el conteo actual de paquetes en la interfaz de red determinada.
  • /conteo-grafica: Muestra una gráfica con el conteo de paquetes cada 5s para la interfaz de red determinada.
  • /estado-interfaz-grafica: Muestra una gráfica con el estado de la interfaz de red cada 5s.
